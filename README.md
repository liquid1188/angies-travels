# Angie's Travels &amp; Tasty Treats

A weekly letter from the road. Destinations, food, recipes, and deals worth your time.

## Live

Deployed via GitHub Pages: https://liquid1188.github.io/angies-travels/

## Stack

Single-file static site. No build step. No framework.

- HTML + CSS + a few lines of vanilla JS
- Fonts: Fraunces (display + body), Inter (UI), JetBrains Mono (metadata)
- Deep blue editorial palette
- IntersectionObserver for reveal-on-scroll
- Sticky navigation
- Fully responsive at 920px breakpoint
